## آدرس سایت

## لیست کارهای انجام‌شده

- **نصب وردپرس**  
- **نصب قالب جدید**  
- **طراحی هدر سایت**  
- **طراحی صفحه اصلی سایت**  
- **ایجاد و طراحی ۲ اسلایدر برای صفحه اصلی سایت**  
- **انتقال کامل محتوا از سایت قدیمی:**  
  - تصاویر  
  - پست‌ها  
  - محصولات  
  - سایر اطلاعات  
- **طراحی و ساخت صفحات زیر:**  
  - درباره ما  
  - ارتباط با ما  
  - محصولات  
  - بلاگ
